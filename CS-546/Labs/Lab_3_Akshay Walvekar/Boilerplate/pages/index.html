Simple Financial Calculator

The below Simple Financial Calculator can be used calculate Retirement Savings, Invested Amount after Certain Period and Number of Months for Loan Repayment

Retirement Savings Amount

Parameter Name Description Example Value
Monthly_Amount This parameter is used for monthly amount of saving Between 0 and 10000
Period This parameter is used for time period of saving Between 0 and 5 years
Interest_Rate This parameter is used for Interest Rate of saving Between 0 and 1

Invested Amount

Parameter Name Description Example Value
Investing_Amount This parameter is used for monthly amount of saving Between 0 and 10000
Period This parameter is used for time period of saving Between 0 and 10 years
Interest_Rate This parameter is used for Interest Rate of saving Between 0 and 1

Months to repay Loan

Parameter Name Description Example Value
Monthly_Amount This parameter is used for monthly amount of saving Between 0 and 10000
Loan_Amount This parameter is used for time period of saving Between 0 and 1,00,000
Interest_Rate This parameter is used for Interest Rate of saving Between 0 and 1
